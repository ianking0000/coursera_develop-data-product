Introduction to MPG-vehicle parameter App

Course project for developing data products

Ian King

Background and Motivation

  1. Mileage per gallon (MPG) is an important factor to consider when buying cars

  2. The App shows relation between MPG and four important vehicle parameters (transmission type, number of cylinders, horsepower and weight). It has two sections.

  3. In section I of the app, you can get plots showing MPG vs. one of the parameters based on statistical data from 32 popular vehicle models

  4. In section II, you can predict MPG by providing these parameters. The prediction is based on optimized linear regression model

  5. This app is based on "mtcars" dataset and optimized linear regression.

Section I: Figure of MPG vs. parameter

Choose one of the four parameters in the side panel, then a figure showing MPG vs. parameter will be present in the main panel.

For example, the plot below is showing MPG vs. transmission type.
The bold horizontal line of each box shows the MPG value of each group, the thin horizontal lines show the MPG range of each group, and the boxsize shows the sqrt of the sample size of each group.

Section II: Predict MPG based on input parameters

Input the four parameters in the side panel, then the predicted MPG value using the input parameters is shown in the main panel.

Input parameter range:
1. transmission type: auto,manual
2. # of cylinders: 4, 6, 8
3. horsepower: 1~500
4. weight(lb/1000): 1~8

The prediction is based on an optimized linear regression model of the mtcars data set (details in next slide).

Overview: dataset and linear regression model

The "mtcar" data set contains mpg value and 10 other key vehicle parameters from 32 automobiles.(from R help)
Using stepwise method in MASS package, it is determined that transmission type, cylinder number, hourse power and weight are most relevant variables for predicting mpg value.
Plot below shows the residuals against the fitted value. It demonstrates the fitting is generally relaible. 

fit4<-lm(mpg~am+cyl+hp+wt,data=ndata)
